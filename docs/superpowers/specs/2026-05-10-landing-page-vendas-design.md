## Decisões de Design

- **Tema:** Dark mode exclusivo, consistente com o app Autchronos
- **Cor de destaque:** Laranja (#f97316) para ênfase textual e bullets
- **CTA:** Verde WhatsApp (#25d366) com glow para destacar o botão principal
- **Tipografia:** -apple-system / BlinkMacSystemFont / Segoe UI (system fonts, sem dependência externa)
- **Seção Agitação:** fundo vermelho escuro para marcar visualmente a virada emocional
- **CTA Final:** gradiente escuro com toque verde para reforçar a cor do WhatsApp
- **Botão WhatsApp fixo na nav:** garante acesso ao CTA em qualquer ponto da leitura
